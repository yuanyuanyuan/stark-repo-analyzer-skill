## 统计口径

覆盖率 = 本次实际请求读取的生产 Python 行范围并集 / 生产 Python 总行数。测试、文档、shell 脚本、配置和外部依赖不计入分母。所有生产文件均按完整行范围读取；大文件按分段请求读取。

## 文件清单口径

生产文件为 `httpx/**/*.py`，排除 `tests/**`。包含 `_transports` 下 6 个文件以及 `__init__.py`、`__version__.py`。未把 `httpcore` 源码纳入本项目覆盖率。

## 限制

源码读取覆盖率不等于运行测试覆盖率，也不等于行为路径覆盖率。本轮没有运行完整测试套件、外部网络请求或 optional extra 组合，因此不能据此声称运行时覆盖率为 100%。
