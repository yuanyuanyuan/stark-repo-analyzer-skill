## 仓库级限制

粗略 `crates/` 非测试/非 benchmark Rust 物理行扫描为 665,641 行。因此本次实际
阅读仅覆盖约 2.3% 的粗略 crates 行数，且该分母包含此次明确排除的 ty、规则、
parser、AST 和其他组件。它不是 standard 模式的“全仓覆盖率”，而是有界核心模块
覆盖率；报告不得把 70.3% 解释为全仓分析完成。
