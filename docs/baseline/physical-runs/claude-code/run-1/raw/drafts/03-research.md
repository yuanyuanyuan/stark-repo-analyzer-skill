## 已验证的定位

README 将该仓库描述为 Claude Code 源码的研究性镜像，并警告它不是官方产品。源码结构显示这是一个 TypeScript/React Ink 终端应用：存在 `src/main.tsx`、CLI handlers、终端渲染层、`QueryEngine`、工具目录、MCP 服务和远程/IDE 连接面。该实现定位仍待模块分析用具体调用链进一步验证。

## 核心问题场景

1. 用户从终端调用一个代理命令时，系统需要在 print/非交互、URI、桥接和常规 CLI 入口之间稳定分流。
2. 交互会话必须同时处理输入、排队回合、流式模型事件、工具调用、权限决策和终端重绘。
3. 外部 MCP 服务会在运行时改变可用工具集合，因此接入层需要把连接、传输、认证/审批和动态刷新组织为明确边界。
