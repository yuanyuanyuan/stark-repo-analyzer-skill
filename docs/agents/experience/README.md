## 使用规则

- 不以目录存在、静态产物或中断运行声称真实 UAT 通过。
- 不将单次固定输入的结果扩大为产品级结论。
- 失败、范围外事项和未满足的验收条件必须保留，不能被后续总结覆盖。
- 新经验必须说明：结论、来源、证据等级、适用范围和反例/边界。
