# Triage Labels

工程技能使用以下 canonical 状态；右列是本仓库 GitHub tracker 的实际 label。

| Canonical role | GitHub label | 含义 |
|---|---|---|
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，可由 AFK Agent 直接领取 |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 不会执行 |

技能提到 canonical role 时，使用对应的 GitHub label，不创建同义重复标签。
