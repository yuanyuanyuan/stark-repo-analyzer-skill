# 大型运行产物不进入 Git

开发验收运行仍须保留足以核验真实 UAT 的原始证据，但完整图谱、Graphify 缓存和长转录存放在 Git 外的本地证据目录或 CI artifact 中。仓库只提交小型 fixture、固定源码 commit、命令与版本、退出状态、耗时、失败分类、结果摘要，以及定位或校验外部原始证据所需的信息，避免运行产物继续主导代码仓库的规模和评审噪声。

## 影响

- 没有原始证据的摘要不能单独证明真实 UAT 通过；外部证据必须在声明通过时仍可定位和核验。
- Graphify 缓存、生成图和完整转录应被忽略或清理，不能因为一次回归成功而加入版本控制。
- 现有历史运行材料需要单独迁移和清理，但清理动作不属于本 ADR 的记录过程。
