# 按失败类别限制 Graphify 重试

> 此决定已被 [ADR-0016](0016-graphify-code-only-v1.md) 取代。以下正文保留历史原貌。

Graphify 只对可识别的暂时性 LLM 故障（网络超时、429、5xx）自动重试，最多两次并采用退避；无可用 LLM、缺少 SDK、不可执行、权限错误、产物缺失、空图和无效 JSON 均立即中止并转人工介入。重试成功后必须重新通过全部图谱产物和证据验证，不能沿用失败尝试的结论。

## 影响

- execution log 和 metadata 必须按尝试记录失败分类、时间、退出码和最终状态。
- 重试预算可预测，不会用重复 LLM 调用掩盖确定性配置问题或消耗失控。
- 验证器和编排器必须共享同一套失败分类，不能由任一方私自降级或继续。
