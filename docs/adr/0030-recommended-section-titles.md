# 终稿推荐章节标题：软推荐；硬判仍靠标记与条目结构

## 决策

- Skill **推荐**稳定中文标题（允许 `## 4. 主链` 类编号前缀）：
  - `主链`
  - `关键设计决策`
  - `风险与改造优先级`
- **structure 硬失败**只依据机器标记与条目结构（main-chain / arch-claim / Risk Item 规则），**不**因标题用词失败。
- 标题偏离推荐集 → 真实UAT **警告**，不单杀 `quality_contract`。
- 风险章节定位可使用标题启发式辅助，但 `risk_count` 计入条件仍以 ADR-0022 为准。
