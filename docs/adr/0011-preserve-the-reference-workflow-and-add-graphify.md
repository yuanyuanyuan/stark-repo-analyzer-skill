# 保真参考流程并加入 Graphify

V1 复制参考 `repo-analyzer` 的分析阶段、Why > What 方法论、源码阅读、subagent 分工、覆盖率门控、交叉验证和报告融合逻辑；唯一的产品增强是将 Graphify 作为阶段 1 与原阶段 2 之间的强制结构证据层。V1 不建立独立分析应用、任务调度服务或新的用户工作流。

## Consequences

- Graphify 只在原流程需要结构导航的位置提供输入：模块候选、问题设计、叙事主线、subagent 上下文、交叉验证和简短报告发现。
- 原有阶段的目标与输出继续有效；Graphify 不能删除或替代源码阅读、用户提问、覆盖率和最终报告责任。
- 改造验收以“参考行为保真 + Graphify 增强生效”为准，而不是以新增运行时代码数量为准。
