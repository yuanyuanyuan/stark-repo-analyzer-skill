# 以规范化后的可用图谱作为健康门

Graphify 健康门保留原始图谱用于诊断，但允许规范化阶段过滤越界、缺失或不可定位的节点和边；只有当过滤后的图谱仍包含非空、来源可定位的节点与关系时才允许继续分析。该决定取代 ADR-0008 中“所有原始节点和边引用必须完整”的严格要求，因为强制门应拒绝没有可用导航证据的运行，而不是让少量 Graphify 噪声阻断整张仍然可用的图。

## 影响

- 规范化必须记录被过滤的节点和边数量，不能静默隐藏 Graphify 质量问题。
- 规范化后没有节点、没有关系或没有可定位来源时，本次分析仍必须终止。
- doctor 只验证规范化产物是否满足可用图合同，不再重复要求原始图零噪声。
