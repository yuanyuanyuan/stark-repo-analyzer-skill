# Archive

本目录保存已经退出活动控制面的历史材料。归档文件可用于理解决策演进和失败边界，但不能覆盖当前 roadmap、exec plan、spec、ADR 或 `docs/dev-rules/`。

- [`v1-control-plane/`](v1-control-plane/)：旧 V1 goals、tasks、计划和输入模板。
- [`v1-execution/`](v1-execution/)：旧 V1 执行记录、问题复盘和 Meta_Kim 经验材料。
- [`local-artifact-index.md`](local-artifact-index.md)：Git 外大型历史工件的位置、规模和校验值；只用于定位与恢复。

归档不是活动任务池。需要恢复其中的事项时，必须在当前 roadmap 或 exec plan 中重新声明。
